 * LINQ
 * SQL Server
 * Redis
 * mongoDb
	
## Launch
to run this project:
```
 $ Clone the repository in your local repo
 $ run database script to create database
 $ open package in visual Studio 2019
 $ Install compatible version of nuget packages
 $ run in web browser
 $ Also Run Redis-server in port 4455 
 
```
